#Bogota Metro

###Prompt:
- Bogota is getting a metro system!

- The program takes the line and stop that a user is getting on at and the line
and stop that user is getting off at and prints the total number of stops for the trip.
- There are 3 metro lines:
  - The Rojo line has the following stops: Calle 62, Calle 42, Carrera 19b, Calle 18, Calle 17, Calle 16, Calle 15, Boyaca, Calle 10, Calle 7
  - The Azul line has the following stops: Calle 63, Calle 45, Calle 26, Centro Historico, Hospitales, Santander, Estacion NQS, Rosario, Avenida 68, Boyaca, Kennedy, Paleneque, Villablanca
  - The Amarillo line has the following stops: Calle 26, Av. NQS, Carrera 14, Carrera 15, Carrera 19b, Carrera 21, Carrera 24, Carrera 26, Portal El Dorado
  - The stations where these lines intersect are Calle 26, Carrera 19b, and Boyaca.

#####Commit 1
- One line functionality
  - Assume that the Rojo line is the only subway line first.
  - The user should be given a list of all the stops on the Rojo line
  - The user should be able to enter the stop that they want to get on at
  - The user should be able to enter the stop that they want to get off at
  - The user should be told the number of stops for their trip

#####Commit 2
- Two lines functionality (Rojo and Azul)
  - The user should be shown lines with their stops listed
  - The user should be able to enter the stop that they want to get on at
  - The user should be given a list of all the stops on that line
  - The user should be able to enter the stop that they want to get off at
  - The user should be told the number of stops for their trip

#####Commit 3
- Three lines functionality
  - Add functionality for the Amarillo line

#####Commit 4 (Bonus)
- Add functionality for a fourth line
  - Call it the Verde line.
  - It must intersect Azul at Centro Historico and Rojo at Calle 17
  - Other stops are up to you!
